# 課程滿意度調查網站

這是一個課程滿意度調查靜態網站，包含課程評分、文字回饋、本機資料保存、表格檢視、CSV 匯出與清除紀錄功能。

## 執行方式

請使用下列指令啟動網站：

```bash
npm run dev
```

網站會啟動在 port `4173`，並綁定 `0.0.0.0`，方便雲端開發環境或預覽工具轉發連線。

> 注意：如果你在遠端容器或線上 IDE 中執行，請使用該平台提供的 Preview / Ports / Forwarded URL。不要直接在自己電腦的瀏覽器輸入 `127.0.0.1:4173`，因為那會連到你自己的電腦，而不是遠端容器。

## Windows 沒有 npm 時

如果 Windows 顯示 `'npm' 不是內部或外部命令、可執行的程式或批次檔。`，代表你的電腦沒有安裝 Node.js/npm。這個網站不一定需要 npm：

1. 下載或解壓縮整個專案資料夾。
2. 直接雙擊 `start-windows.bat`。
3. 或直接雙擊 `index.html`。

這會用瀏覽器直接開啟靜態網站，不需要安裝 Node.js。

## 不用伺服器，直接打開檔案

如果 Preview / Ports 無法使用，這個網站也可以不用伺服器直接開啟：

1. 下載或複製整個專案資料夾到你的電腦。
2. 在檔案總管中雙擊 `index.html`。
3. 瀏覽器網址會顯示類似 `file:///.../index.html`，表單、localStorage 與 CSV 匯出仍可使用。

請不要在自己的瀏覽器輸入 `127.0.0.1:4173` 來連遠端容器；`127.0.0.1` 永遠代表你目前正在使用的那台電腦。

## 直接用 Python 執行

也可以用 Python 直接提供靜態檔案：

```bash
python3 -m http.server 4173 --bind 0.0.0.0
```
